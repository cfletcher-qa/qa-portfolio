# API Testing

This folder contains examples of manual and automated API testing using public APIs.

## Tools Used
- Postman
- REST APIs (public test APIs)
- JSON Schema validation

## Coverage
- GET, POST, PUT, DELETE requests
- Positive and negative test scenarios
- Status code validation
- Response body validation
- Query parameters
- Authentication (Bearer token examples)

## Purpose
Demonstrates API testing techniques commonly used in QA interviews and real testing environments.
